# Screenshots Directory

Place your application screenshots in this directory.

## Recommended Screenshots

1. **sign-in.png** - Sign in/Sign up screen
2. **player-registration.png** - Player registration form
3. **availability-poll.png** - Weekly availability poll interface
4. **team-results.png** - Generated teams display
5. **admin-panel.png** - Admin user management interface

## Screenshot Guidelines

- **Format**: PNG or JPG
- **Size**: Recommended 1920x1080 or similar aspect ratio
- **Quality**: High resolution for clarity
- **Content**: Show the main features and UI elements clearly

1. Run the app locally: `npm start`
2. Navigate to each section
